This is the first post on a new home for the work I've been doing around AI, Claude Code, and the practice of building software with agents in the loop.

For a while now I've been publishing skills, recipes, and patterns to a public GitHub repo as plain markdown. That worked, but markdown viewed in GitHub's file browser is not where this content wants to live. So: a real site, a real domain, a real place to write.

## What you'll find here

- **Recipes** — small, reusable patterns for getting work done with Claude Code
- **Skills** — the toolkit I install into every project, with notes on when each one earns its keep
- **Essays** — longer-form thinking about agent-shaped workflows, AI adoption, and how teams actually change

The plugin and the source markdown still live in the same repo — `github.com/anutron/ai`. This site is just a friendlier front door.

More soon.
