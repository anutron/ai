The skills directory is the heart of the A-stack. Each skill is a focused, self-contained capability that Claude Code can invoke when relevant — or you can call directly with `/skill-name`.

Highlights:

- **brainstorm** — explore intent before you write code
- **execute-plan** — turn a plan into parallel agent-driven implementation
- **bugbash** — iterative QA session with auto-dispatched fix agents
- **plannotator-review** — interactive code review in the browser
- **ralph-review** — spec-driven implementation review
