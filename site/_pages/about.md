I'm Aaron Newton — VP Product at [Thanx](https://thanx.com), and a long-time builder, illustrator, and tinkerer.

This site collects the work I do around AI-augmented software development:

- **Skills, hooks, and patterns** for [Claude Code](https://claude.com/claude-code), packaged as a plugin you can install
- **Recipes** — small, reusable patterns for getting work done with agents in the loop
- **Essays** on adoption, workflow, and the practice of building with AI

Everything here is open and shareable. The plugin and source markdown live at [github.com/anutron/ai](https://github.com/anutron/ai).

If you want to install the plugin in Claude Code:

```
/plugin install ai@anutron/ai
```

Find me on [GitHub](https://github.com/anutron) or reach out at [aaron@anutron.com](mailto:aaron@anutron.com).
